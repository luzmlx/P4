## Overview

**Sweet Treats** is a bakery application where you can add a cake item and update it.


<br>

## MVP



MVP 

  -Full CRUD
  
  -Create User, Sign In, Sign Out
  
  -Build Ruby on Rails server
  
  -Media Query
  
  -Responsive Buttons


### Goals

- Reach MVP and have a working, responsive web application.
